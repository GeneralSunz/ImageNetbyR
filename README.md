# ImageNet 极简版图像分类 (R + Python)

使用预训练 MobileNetV2 模型对本地图片进行 ImageNet 分类。

## 文件结构

```
├── image_classfication.r    # R 脚本
├── image_classification.py  # Python 脚本
├── book_images/              # 存放待分类图片
└── README.md
```

## 环境要求

- Python 3.13+ / R >= 4.4
- Python 包：`keras`, `pillow`, `numpy`（脚本自动安装）
